UNITY PROJECTS

Ikki's Adventure - Infinite Runner

This game was made in collaboration with some colleagues from university.

This was our first game and at the time all of us were addicted to an infinite runner game on Facebook, so we thought: "what if we create our own infinite runner game?".

And that's how it all started. We had never touched on Unity before but, since it was very popular, we decided to give it a try and we enjoyed it a lot.

In this game, I was responsible for the art (scenario, characters, animation, etc.) and some coding parts.

Recycling Game - Match 3

This game was also made in collaboration with some colleagues from university.

The intention is to teach how to throw the garbage to the correct bin but in a fun way, so that kids would enjoy playing and at the same time they would also learn.

Once again, I was responsible mainly for the art.
